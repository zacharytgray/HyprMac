Direction & reasoning
Accent semantics, made systemic: cyan = focus & active (focus border, brackets, active workspace, toggles-on, primary buttons). Magenta earns one job: the floating layer — floating-window borders, scratchpad scrim + tray, and every ◇ floating marker. Two colors, two meanings, everywhere.
Settings IA: 4 tabs → 3. General · Keys · Layout. App Launcher merges into Keys as an "Apps" category (a launcher is a keybind). Focus indicator + dimming live in Layout with the gaps + per-monitor controls.
Flows: onboarding / welcome / what's-new collapse into one Tour shell — same window, same header/footer anatomy; only the page set differs per trigger.
Hypr+K: three explorations below — refined list (1g), keyboard map (1h), dense mono sheet (1i).
Chrome defaults: focus color defaults to hyprCyan (not system accent), overridable. One shared motion table for border / brackets / dim / scrim.
